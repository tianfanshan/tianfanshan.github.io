Generate and download PDF file

2023-04-03

How to generate a web page to PDF file

1.Install a NuGet Package names Rotativa to the project
2.Download wkhtmltopdf.exe from the internet to the wwwroot folder, it helps us to convert the html file into PDF
3.Create a new view for generate it as PDF, the reason is that, we don’t want to convert the entire page with the _layout view


4.Add a link to the index view, when we click on it will active a method of controller to generate the PDF

5.Add a new method in the controller to receive the request from index view, at there we call the database to fetch a list of PersonResponse, then it will go to the PersonsPDF view along the data

6.As we only want the list of persons to print on the PDF file, at this file there is no other datas apart from the list of persons

7.Start the project, we can get to the main page, and there is the button for download the PDF file

8.As you can see, we only generate the list of those persons nothing else, and it also can be downloaded
