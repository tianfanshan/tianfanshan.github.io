Generate CSV file

2023-04-04

Step by step to generate CSV file on a ASP.NET Core web application

1.Create a new method in the services through the interface. It returns Task<MemoryStream>**, we can easily convert the same as file in the controller action method, because MemoryStream represents a stream of bytes. At line 252-257 is writing the header of the column, and at line 269-281 is writing the list of persons, even you can add more informations or less, it depends on you.


2.Install CsvHelper NugetPackage into that project, to help us to generate the CSV file.

3.In controller we have to create a new response method to provice the CSV file back to the client with CSV’s mime type(application/octet-stream), and the name of that file.

4.At the main page we created a link to active that method to generate the CSV file along the list of persons.

5.Once clicked that button the CSV file will be downloaded.
