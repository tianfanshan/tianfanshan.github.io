Generate Excel file

2023-04-04

Generate Excel from web page file step by step

1.Install EPPlus Nuget package in the services project


2.Set EPPlus package for non-commercial usage in the appsettings.json file,
when LicenseUsageMode is set to nonCommercial, if the application is used for commercial purposes, an exception will be thrown to prevent unauthorized commercial use.

3.Create a new method for generating Excel file, in services and the interface. Line 293 create a new MemoryStream object to store the data from database. Line 294-296 create a ExcelPackage and ExcelWorksheet(from EPPlus) object to create, read or edit the excel file. Line 297-304 to generate the header for each column. Line 307-312 to give that Excel file a style. Line 314-330 to fetch the data from database and write those data in the Excel file. Line 331 to automatical adjust the width of all columns in a worksheet to fit their longest cell contents. Line 334 write the excel into the memory stream. Finally we renturn that memory stream back to the client.

4.Call that method from controller to return that MemoryStream object as the mime type application/vnd.openxmlformats-officedocument.spreadsheetml.sheet.

5.At the main page we have a button to generate the page as Excel file.

6.Downloaded Excel file.
