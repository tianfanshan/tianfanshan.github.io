ILogger and HTTP logging

2023-04-26

Where could we use ILogger and why

We can use ILogger to print messages on the console to help us to debugging or tracking the execution sequence of the application.

Inject ILogger into the classes we want, create a field and inicialize it in the constructor, at the filed we have to pass the current working class name as generic parameter.


Use the ILogger object’s method to print the message wherever we want in the method.

It also can be used in other classes, such like repository, service etc…

What is the usage of HTTP logging

The HTTP logging refers the process of recording HTTP requests and responses between a client and server. The primary purpose of HTTP logging is to gather information about web traffic for analysis, troubleshooting, and security purposes.

To enable HTTP logging we have to add it to the request pipeline(program.cs) after the builder.Build() method, and write app.UseHttpLogging() to enable it.


We also can configure the HTTP logging to log only the specific details, we can add it above the builder.Build() method of Program.cs as a service and by using the lambda expression to define which messages we want to print(Line 38 - 42):

There are more method of Microsoft.AspNetCore.HttpLogging.HttpLoggingFields we can use:
1. RequestMethod: Method of request
2. RequestPath: Path of request
3. RequestProtocol: Protocol of request
4. RequestScheme: Protocol Scheme of request
5. RequestQuery: Query string Scheme of request
6. RequestHeaders: Headers of request
7. RequestPropertiesAndHeaders: Includes all of above(default)
8. RequestBody: Etire request body(has performance drawbacks, it is not recommended)
9. Request: Includes all of above
10. ResponseStatusCode: Status code of response
11. ResponseHeaders: Headers of response
12. ResponsePropertiresAndHeaders: Includes all of aboce(default)
13. ResponseBody: Entire response body(has performance drawbacks, it is not recommended)
14. Response: Includes all of above
15. All: Includes all from request and response
