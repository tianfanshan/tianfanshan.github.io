Customized query for ProductRepository

  • In the ProductRepository simply add a customized query by using the annotation @Query, then we can add the query we want as an attribute in it. Instead of using products table at here we use the Entity name Product, then we can pass the parameter from request in it too.

Product Entity

  • SKU stands for Stock Keeping Unit.
  • In the Entity we implement 2 new annotations:
    1. @CreationTimestamp: whenever we save a entity, hibernate will automatically assign the timestamp for the property.
    2. @UpdateTimestamp: whenever we update a entity, hibernate will automatically assign the value for the property.

The rest are ordinary springboot application codes, nothing new.
