How to publish a app as folder

2023-02-24

Why do we have to publish our project

Because our project is made to run on our machine, we dont know about other people that machine they use. We have to make sure that our project can run on them, for example: Linux, Mac or Windows. Once we publish the project they can run it on most system and most environment.

Right click on you project select publish


Select the place you want to publish

Select the local target would you like to publish to

Select the path to a local or network folder

Click publish

Run application on terminal

If i give someone that folder and they have Windows, they don’t have to install anything else, the can just copy it and run it anywhere.

Copy the path where are the published files


Cd the the direction that we copied and run the MySuperBank.exe
