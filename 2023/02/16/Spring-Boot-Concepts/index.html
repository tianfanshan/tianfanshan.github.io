What is Spring Boot

Spring boot is basically an extension of Spring framework, it is builted on top of spring framework to reduce a lot of configuration required to set up the spring based applications.
Spring boot allows the developer to quickly create production Spring-based applications without requiring developers to werite the same boilerplate configuration again and again.

A few important Spring Boot features

Spring Boot Starters
It offers many starter models to getting starter quickly: Spring MVC, JPA, MongoDB, Spring Security etc…
We no longer have to search for compatibility library version and configure them manually.

Spring Boot Autoconfiguration
It attempts to automatically configure a Spring Application based on the jar dependencies that you have added.
Without is we have to configure a lot things:
When we use Spring MVC: Component Scan, Dispatcher Servlet, View resolver and Web jar
When we use Hibernate/JPA: Dtasource, Entity manager factory/session factory and transaction manager amon a host of other things

Externalized Configuration
It enables developer to work with the same application in different environment: dev, prod, qa etc…
It allows us to use properties file to mantain the configuration.

Spring Boot Actuator
Spring Boot provides a wide variety of production ready features without requiring developers to write a much code, we can use it to view:
The application bean configuration details
The application URL mappings
The environment details
Configuration parameter values
The registered health check metrics

Easy-to-use embedded servlet container support
We can quickly run our Spring application in a better servlet container, because it by default provides Tomecat server as a default servlet container, so we no longer haver configure the Tomcat server
