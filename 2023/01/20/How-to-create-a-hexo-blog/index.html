How to create a hexo blog

2023-01-20

This is my first blog, and i have thank to CodeSheep to help to create it. And here the video from youtube How to create yourself blog

Environment

  • Node.js
  • npm

How to set hexo

  1. Use npm to download hexo package with command line: (npm install -g hexo-cli)
  2. Command line: (hexo -v) to check if the hexo is installed successfully
  3. Create a directory (dir name what you want): blog
  4. Command line: (hexo init), it will create all the stuff you need for the blog in the dir

How to create a blog

  1. Use command line: (hexo n “blog title”) to create a new blog.mb
  2. You can use Visual Studio Code to write your blog
  3. Once you finished writing your blog at first use command line: (hexo clean) to clean -> (hexo g) to generate -> (hexo s) to start

How to deploy to github

  1. Create a repository with your account name, for example: tianfanshan.github.io
  2. Back to command line use: (npm install –save hexo-deployer-git)
  3. Set _config.yml file, there is a deploy field:
    deploy:
    type: git
    repo: your-repository-HTTPs
    branch: master
  4. Deploy your blog to github with command line: (hexo d)
  5. Maybe you have to log in you github account on terminal
  6. Then you can access your blog on line by searching the blog repository name on url, for example: tianfanshan.github.io

How to change your blog themes

  1. Find a hexo blog theme you like on github
  2. Use command line to clone it to your blog directory: git clone (repository of themes) themes/(dirName)
  3. Set _config.yml file, there is a them field:
    theme: dirName
  4. Use command line to see your blog locally: hexo clean -> hexo g -> hexo s
  5. Use command line to push your new blog to git hub

Remove post

  1. Go to blog/source/_posts, and delete the post you want
  2. Command line: hexo clean -> hexo g -> hexo d
