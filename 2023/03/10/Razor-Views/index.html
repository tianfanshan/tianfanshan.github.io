Razor Views

2023-03-10

What is Razor view

It is a view engine used in the ASP.NET web application framework for building web pages. It allows developers to create dynamic HTML pages by combining HTML markup with server-side code written in C#.

How does it work

  1. When the client send request from the browser, it will search the url route in the controller, then return a cshtml as a view along the data back to the client
  2. It will search the file in the Views folder, and according the prefix of the controller here is Home, and finally find the file that named by using the method name in the controller
  3. In this .cshtml file we can use that data we passed from the controller to show it up on the browser, at file we can write C# in it by using @ before the logic
  4. By the way if you want to send more than one Model then we can create another model including those models, such like in the image
  5. Another thing to metion up is that, we can create a _ViewImports.cshtml file in the folder of the /Views or in the Views/Home or what ever to import the name space for those views below it

Summary

For me it works like Java servlet, nothing amazing, and I believe this should be obsolete technology, because there we are working the logic together with static HTML file, i think that is not recommended in the present era.
