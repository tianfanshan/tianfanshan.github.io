Browse the application on mobile locally

2023-06-01

To test the springboot web application locally on mobile

  1. Connect the mobile and the computer to the same network(LAN)
  2. Open the command prompt on the computer, type ipconfig on C drive level to know the Wireless LAN adapter Wi-Fi IPv4 address
  3. On the browser of the mobile enter http://IPv4Address:8080
