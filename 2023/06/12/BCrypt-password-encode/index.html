BCrypt password encode

2023-06-12

What is BCrypt

BCrypt is a password-hashing function based on Blowfish(symmetric-key block cipher) and crypt function in Unix.

It is high secure because it has a salt used to protect against rainbow table attacks.
It is adaptive, strength can be increased when hardware gets faster slow down attackers.

BCrypt is a widely-used password hashing algorithm designed to securely store passwords. It is specifically designed to be slow and computationally intensive, which makes it resistant to brute-force and dictionary attacks. The BCrypt algorithm incorporates salt and multiple rounds of hashing to enhance password security.

How does it work

  1. Generate a random salt: A salt is a randomly generated value that is combined with the password before hashing. It adds uniqueness to each password hash, even if two users have the same password. The salt is typically stored alongside the hashed password.
  2. Hash the password: The password, along with the salt, is hashed using the BCrypt algorithm. The algorithm repeatedly applies a key setup phase, which includes mixing the password and salt, and then performs a specified number of iterations to produce the final hash.
    3.Store the hashed password and salt: The resulting hash, along with the randomly generated salt, is typically stored in a database. The salt is stored alongside the hash so that it can be used during password verification.

Summary

The main advantage of BCrypt is its slowness, which makes it computationally expensive for an attacker to perform large-scale password cracking attempts. Additionally, BCrypt automatically handles the generation and verification of the salt, making it easier for developers to use.

It’s important to note that BCrypt is just one of many password hashing algorithms available, and it’s always recommended to use a well-vetted and up-to-date algorithm for password storage.
